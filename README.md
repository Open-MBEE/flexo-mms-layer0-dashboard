# MMS5 Layer 0 Admin Dashboard
Webapp GUI allowing admins to inspect and edit MMS5 named graphs.

## Install

For NPM users:
```sh
npm i
```

For Yarn:
```sh
yarn install
```

For NPM users:
